# デスクトップ時計アプリ「yClocky」要件定義書

## 1. プロダクトコンセプト
*   **「究極にミニマルなデスクトップ時計」**
*   OSの標準機能のようにデスクトップに溶け込み、存在を主張しすぎない。
*   余計なアイコンやメニューを一切表示せず、必要な操作はショートカットで行う。

## 2. デザイン・表示機能 (UI)

*   **基本表示:**
    *   デジタル時計（時・分・秒）。
    *   日付、曜日の表示（ON/OFF可）。
*   **カスタマイズ:**
    *   **フォント:** PC内のインストール済みフォントを自由に選択可能。
    *   **配色:** 文字色、背景色の自由変更。
    *   **透明度:** 背景および文字の透明度（Alpha値）を個別に調整可能。
*   **高DPI対応:**
    *   WPFのベクター描画特性を活かし、4KモニターやWindowsの拡大縮小設定（100%〜200%等）変更時でも、文字が美しくスケーリングされ、レイアウトが崩れないこと。

## 3. ウィンドウ制御・挙動 (UX)
本アプリの最大の特徴である「ステルス性」と「直感的なアクセス」を定義する。

*   **完全なリスト除外 (Stealth Mode):**
    *   **`Alt + Tab` 非表示:** アプリケーション切り替え画面に表示しない（ToolWindowスタイルの適用等）。
    *   **タスクバー非表示:** 画面下のタスクバーにアイコンを表示しない。
    *   **タスクトレイ非表示:** 通知領域にもアイコンを表示しない。
*   **設定画面へのアクセス:**
    *   **`Ctrl` + `Click` (左クリック):** 時計ウィンドウをクリックすることで設定画面を呼び出す。
    *   （※誤操作防止のため、単なるクリックやダブルクリックではなく修飾キーを必須とする）
*   **表示レイヤー:**
    *   **常に最前面に表示 (Always on Top):** 他のウィンドウよりも手前に表示する設定（ON/OFF可）。
*   **位置調整:**
    *    `Ctrl` + ドラッグによる移動
    *   再起動後も前回終了時の位置を正確に復元する。
    *   マルチモニター環境での位置保持。

## 4. 開発技術 (Technical Stack)
Windows環境における親和性と描画品質、ウィンドウ制御の柔軟性を確保するため、以下の技術を採用する。

*   **言語:** C#
*   **フレームワーク:** WPF (Windows Presentation Foundation) / .NET 8
*   **配布形態:** シングルファイル（Self-contained）での配布を想定
